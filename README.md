# EduFlow

نظام منصة تعليمية بسيط يعمل من ملف `index.html` مباشرة، ويدعم:

- إدارة الطلاب وتوليد QR لكل طالب.
- إدارة الكورسات والمدرسين والمواعيد.
- تسجيل حضور يومي عبر مسح QR.
- تسجيل الدرجات وإرسال إشعارات واتساب.
- استخراج قائمة الغياب والتنبيه لولي الأمر.

## التشغيل

افتح `index.html` في المتصفح مباشرة.
